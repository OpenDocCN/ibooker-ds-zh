# 第三部分. 案例研究 3：使用新闻标题追踪疾病爆发

## 问题陈述

恭喜！您已被美国健康研究所录用。研究所监控国内外疾病流行情况。监控过程中的一个关键组成部分是分析发布的新闻数据。每天，研究所都会收到数百条描述各种地点疾病爆发的新闻标题。新闻标题太多，无法手动分析。

您的第一个任务是：您将处理每日的新闻标题配额，并提取提到的地点。然后，您将根据地理分布对标题进行聚类。最后，您将审查美国境内外的最大聚类。任何有趣的发现应报告给您的直接上级。

### 数据集描述

文件 headlines.txt 包含您必须分析的数百条新闻标题。每个标题都在文件的单独一行上。

## 概述

为了解决当前的问题，我们需要知道以下内容：

+   使用多种技术和距离度量对数据集进行聚类。

+   测量球面上各地点之间的距离。

+   在地图上可视化地点。

+   从新闻标题文本中提取地点坐标。
