## 下载

### Docker

```
docker pull apachecn0/ds-cmd-line-2e-zh
docker run -tid -p <port>:80 apachecn0/ds-cmd-line-2e-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g ds-cmd-line-2e-zh
ds-cmd-line-2e-zh <port>
# 访问 http://localhost:{port} 查看文档
```
