# 第四部分：领导者：激励一个行业

成功的 DS 领导者和杰出数据科学家散发出超越他们公司的魅力。他们通过产生高度评价的成就来展示 DS 的影响并激励他们的行业。他们以冷静自信的态度行事，这导致深思熟虑和及时的计划与行动，专注于从其组织中发挥最佳作用。

领导 DS 的领导者可能拥有首席数据科学家、首席数据官或数据副总裁等头衔。有时，DS 团队向首席技术官、首席财务官或首席风险官汇报。个人贡献型领导者通常拥有杰出数据科学家或 DS 研究员等头衔。无论报告结构如何，DS 领导者或杰出数据科学家都应该了解领导者的属性和实践，以确保 DS 功能的成功。

领导数据科学（DS）的领导者所具备的能力包括公司内部和外部责任。这个范围比董事的职责更广，董事主要负责在公司内部治理 DS 功能。

作为内部关注点，DS 领导者或杰出数据科学家负责公司的长期可持续性。这包括制定多年的商业战略，明确 DS 使命，在各个职能中培养数据驱动文化，捕捉新兴的商业机会以实现业务目标，以及为新数据驱动产品线阐述商业计划。

作为外部关注点，他们负责阐明业务模式在行业同行中的竞争优势，建立强大的人才品牌和招聘渠道，以及学习各行业的最佳实践。虽然内部和外部举措都可以委托，但这些举措的倡导者和推动者应该是你，DS 领导者。

要练习领导力，你必须将个人激情转化为为他人创造积极差异，练习积极的思维模式，通过有效的情绪模式保持领导者的风度，以及与团队一起展现有效的行动模式。

第八章将介绍 DS 领导者和杰出数据科学家的能力，讨论在技术、执行和专业知识方面的内部和外部责任。第九章将通过讨论实践 DS 的道德、严谨和态度来阐述领导者的美德。
