# 第五部分. 循环与未来

作为一位高绩效的 DS 从业者，你在分析严谨性方面有天赋。如果你能将其应用于你的职业生涯呢？你能更好地释放你的潜力吗？你将如何应用你的分析严谨性来自信地推进你的职业生涯？

你可以查看*LOOP*，它包括景观、组织、机遇和实践这四个领域。这些领域形成一个循环，因为你可以每月回顾一个领域，以在你的环境中获得新的解释，评估你可以为你的职业生涯做更多的事情。

理解技术*格局*可以帮助你抓住正在改变你行业的下一波机遇。对你项目背后的**组织**和倡议的演变有所了解，可以帮助你更好地导航团队和职能的运营动态。一个稳健的入职计划对你可获得的*机遇*进行评估，可以帮助你自信地承担新的责任。了解 DS 中可用的*实践*可以帮助你识别你可以雇佣的技能组合以及你可以选择的职业方向。

DS 领导具有挑战性，因为它涉及一组广泛的技能，这些技能需要时间来内化。这些技能是由与数据合作以及协调高管、团队和合作伙伴以使 DS 工作成功的独特挑战塑造的。

我们总结了 DS 领导中的*为什么*、*是什么*和*怎么做*，以便你在职业生涯的各个阶段进行参考。我们还将展望未来，对领域在角色、能力和责任方面的可能演变进行一些推测，以预测。

第十章提供了一个起点，通过景观、组织、机遇和实践（或 LOOP）的视角严格审视你的环境。第十一章总结了 DS 领导背后的深层动机，并对可能从根本上塑造未来领域的趋势进行了推测。
